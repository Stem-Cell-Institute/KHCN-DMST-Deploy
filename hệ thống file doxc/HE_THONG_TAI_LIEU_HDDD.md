## 🔄 QUY TRÌNH SỬ DỤNG

### Giai đoạn 1: Chuẩn bị (Nghiên cứu viên)
```
1. Đọc hướng dẫn → 00_MUC_LUC + HUONG_DAN_TONG_HOP
2. Điền 3 mẫu bắt buộc → SCI-ACE-01, 02, 03
3. Chuẩn bị tài liệu đính kèm
4. Nộp hồ sơ
```

### Giai đoạn 2: Đánh giá (Hội đồng)
```
1. Tiếp nhận hồ sơ (Thư ký)
2. Phân công phản biện (Chủ tịch)
3. Đánh giá → SCI-ACE-PĐG (Thành viên)
4. Họp thẩm định → SCI-ACE-05 (Biên bản)
5. Quyết định → SCI-ACE-QĐ (VN + EN)
```

### Giai đoạn 3: Thực hiện (Nghiên cứu viên)
```
1. Nhận Quyết định
2. Thực hiện theo đề cương
3. Có thay đổi? → SCI-ACE-06
4. Báo cáo định kỳ/kết thúc → SCI-ACE-07
```
