Nộp hồ sơ bổ sung (Bước 2)

Quá trình vẫn đang ở Bước 2. Hồ sơ đã nộp (gốc và bổ sung) hiển thị bên dưới. Nếu cần, thêm file bổ sung qua các ô tải lên và nhấn Gửi hồ sơ bổ sung. Thư ký sẽ kiểm tra lại và nhấn Hợp lệ hoặc Yêu cầu bổ sung tiếp.

📎 Hồ sơ bổ sung (lần mới)

Thêm file (PDF, DOCX). Nhấn dấu + để thêm ô tải lên.

Nộp hồ sơ đề tài cấp Viện

Không gian nộp hồ sơ thế hệ mới: điền thông tin, tải file bắt buộc, đính kèm tài liệu liên quan và gửi trong một lần. Sau khi nộp, bạn theo dõi toàn bộ tiến độ tại mục "Hồ sơ chi tiết".

Tiến độ hoàn tất hồ sơ 0%
Bước 1Nhập thông tin đề tài
Bước 2Tải đủ 3 file bắt buộc
Bước 3Sẵn sàng gửi hồ sơ

1 Thuyết minh đề tài

Mô tả mục tiêu, nội dung, phương pháp và kết quả dự kiến. * Bắt buộc.

Kéo thả file vào đây hoặc bấm để chọn file

2 Kế hoạch thực hiện

Tiến độ, nhân lực, kinh phí dự kiến. * Bắt buộc.

Kéo thả file vào đây hoặc bấm để chọn file

3 Tài liệu bổ sung

Lý lịch khoa học, cam kết (nếu có). * Bắt buộc.

Kéo thả file vào đây hoặc bấm để chọn file

Tài liệu đính kèm (tùy chọn)

Kéo thả nhiều file hoặc bấm để chọn

Hệ thống kiểm tra tự động: 3 tài liệu bắt buộc + tên đề tài trước khi cho phép gửi.

Chưa đủ điều kiện gửi hồ sơ.
